# Public Sources

This directory is reserved for downloaded public and official source materials used by the implementation.

Expected contents:

- a small committed example subset of HTML or JSON source files
- `manifests/registry_snapshot.json`
- `manifests/download_manifest.json`

The current committed example subset includes:

- one regulatory guidance source
- one ClinicalTrials.gov study record

The manifests describe a larger downloadable public-source bundle than the small subset committed here. If redistribution is uncertain, keep only the manifests and regenerate the source files locally with the ingestion commands described in the top-level README.
